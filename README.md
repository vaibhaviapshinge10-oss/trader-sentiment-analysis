# Overview
This project analyzes how trader performance and behavior change under different market sentiment regimes (Fear vs Greed index).

It includes performance analysis, behavioral patterns, segmentation, and clustering of traders based on trading activity and risk profile.
# Required Data Files
To run this notebook, you must upload the following two datasets (provided in the assignment):

Historical Trader Data

Fear & Greed Index Data

The notebook expects both files to be uploaded manually in the runtime environment.

# How to Run (Google Colab)

1. Open the notebook from this repository.

2. Click “Open in Colab” (or upload the notebook to Google Colab manually).

3. Upload the two required data files when prompted.

4. Run all cells from top to bottom.
